# Contributing to Playtime Tracker

Thank you for helping improve Playtime Tracker. Contributions are welcome from anyone who wants to help make the app more reliable and polished.

## Setup

1. Clone the repo:
   ```bash
git clone https://github.com/yourname/playtime-tracker.git
cd playtime-tracker
```
2. Create and activate a virtual environment:
   ```bash
python -m venv .venv
.\.venv\Scripts\activate
```
3. Install dependencies:
   ```bash
pip install -r requirements.txt
```
4. Run tests:
   ```bash
pytest
```

## Coding style

- Follow PEP 8 formatting and naming conventions.
- Keep type hints consistent and complete.
- Add docstrings for public classes and important methods.
- Keep behavior unchanged unless implementing a deliberate improvement.

## Contributions

- Open a new issue for bugs or feature requests.
- Submit a pull request with a concise summary of your changes.
- Include tests for any new functionality or bug fix.
- Describe how to reproduce the issue or verify the fix.
